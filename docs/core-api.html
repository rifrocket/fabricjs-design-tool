üé®Canvas Management

useCanvasManager(canvas, options?)

Main hook for managing canvas state and operations.

useCanvasManager(canvas: fabric.Canvas, options?: CanvasManagerOptions)

Parameters

Parameter Type Description
canvas fabric.Canvas Fabric.js canvas instance
options CanvasManagerOptions Optional configuration options

Returns

{
  canvas: fabric.Canvas | null;
  selectedObject: fabric.Object | null;
  zoom: number;
  isDarkMode: boolean;
  alignmentGuides: AlignmentGuidesConfig;
  
  // Methods
  selectObject: (object: fabric.Object) => void;
  deleteSelected: () => void;
  duplicateSelected: () => void;
  setZoom: (zoom: number) => void;
  resetZoom: () => void;
  fitToScreen: () => void;
  toggleDarkMode: () => void;
}

Example

import { Canvas } from 'fabric';
import { useCanvasManager } from 'fabricjs-design-tool';

const canvas = new Canvas('my-canvas');
const canvasManager = useCanvasManager(canvas, {
  showGrid: true,
  gridSize: 20,
  alignmentGuides: true
});

// Select an object
canvasManager.selectObject(someObject);

// Zoom controls
canvasManager.setZoom(1.5);
canvasManager.fitToScreen();

Enables pan functionality with mouse/touch.

Example

const panControls = useCanvasPan(canvas);

// Pan is automatically enabled
// Hold Shift + drag to pan (configurable)

useCanvasKeyboardShortcuts(canvas)

Adds keyboard shortcuts to canvas.

Default Shortcuts

  • Ctrl+Z / Cmd+Z - Undo
  • Ctrl+Y / Cmd+Y - Redo
  • Delete / Backspace - Delete selected
  • Ctrl+C / Cmd+C - Copy
  • Ctrl+V / Cmd+V - Paste
  • Ctrl+A / Cmd+A - Select all
  • Escape - Deselect all

üîßShape Factory

shapeFactory

Factory methods for creating shapes with consistent styling.

Available Methods

Method Description
createRectangle(options) Create a rectangle shape
createCircle(options) Create a circle shape
createTriangle(options) Create a triangle shape
createLine(options) Create a line shape
createText(text, options) Create a text object
createArrow(options) Create an arrow shape

Example

import { shapeFactory } from 'fabricjs-design-tool';

// Create a rectangle
const rect = shapeFactory.createRectangle({
  left: 100,
  top: 100,
  width: 200,
  height: 150,
  fill: 'blue',
  stroke: 'red',
  strokeWidth: 2
});

// Create a circle with gradient
const circle = shapeFactory.createCircle({
  left: 300,
  top: 100,
  radius: 75,
  fill: new fabric.Gradient({
    type: 'radial',
    coords: { x1: 0, y1: 0, x2: 0, y2: 0, r1: 0, r2: 75 },
    colorStops: [
      { offset: 0, color: '#ff0000' },
      { offset: 1, color: '#0000ff' }
    ]
  })
});

canvas.add(rect, circle);

üõ†Canvas Utils

canvasUtils

Utility functions for canvas operations.

Available Methods

Method Description
exportCanvas(canvas, format, options) Export canvas to various formats
loadFromJSON(canvas, json) Load canvas from JSON data
saveToJSON(canvas) Save canvas to JSON
clearCanvas(canvas) Clear all objects from canvas
centerObject(canvas, object) Center an object on canvas

Export Formats

import { canvasUtils, EXPORT_FORMATS } from 'fabricjs-design-tool';

// Export as PNG
const pngDataUrl = canvasUtils.exportCanvas(canvas, EXPORT_FORMATS.PNG);

// Export as PDF
const pdfBlob = canvasUtils.exportCanvas(canvas, EXPORT_FORMATS.PDF, {
  format: 'a4',
  orientation: 'portrait'
});

// Export as SVG
const svgString = canvasUtils.exportCanvas(canvas, EXPORT_FORMATS.SVG);

üìùHistory Manager

historyManager

Manage undo/redo operations.

Methods

Method Description
init(canvas) Initialize history tracking
undo() Undo last action
redo() Redo last undone action
canUndo() Check if undo is available
canRedo() Check if redo is available
clear() Clear history

Example

import { historyManager } from 'fabricjs-design-tool';

// Initialize history
historyManager.init(canvas);

// Use undo/redo
if (historyManager.canUndo()) {
  historyManager.undo();
}

if (historyManager.canRedo()) {
  historyManager.redo();
}

üìèAlignment Guides

useAlignmentGuides(canvas, options?)

Enable smart alignment guides for precise object positioning.

Options

Option Type Default Description
enabled boolean true Enable/disable alignment guides
snapDistance number 10 Snap distance in pixels
lineColor string '#ff0000' Guide line color
lineWidth number 1 Guide line width

Example

import { useAlignmentGuides } from 'fabricjs-design-tool';

const alignmentGuides = useAlignmentGuides(canvas, {
  enabled: true,
  snapDistance: 15,
  lineColor: '#00ff00',
  lineWidth: 2
});
